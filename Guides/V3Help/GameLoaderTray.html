GameLoader Tray

Function

Its an application that loads an icon in the system tray from which a menu can be accessed which allows the user to open GameManager, Database Manager and Help files. Its a small application that uses few system resources hence it doesn't impact on system performance.

Program Interface

System tray icon looks like the circled icon below:




When you right click you get a menu like below




Open Game Manager – Runs Frontend of GameManager
Update Database – Runs DatabaseManager
Setup – Runs the setup program so you change the program options
About – Tells you about the program
Help – Opens web browser with these help documents
Exit – Closes the tray application
